# AWS EC2 Instance Deployment

This project demonstrates launching an Amazon EC2 instance in AWS and understanding basic instance configuration and networking.

## Project Objective

To understand the basics of:

- Amazon EC2
- Security Groups
- Key Pairs
- Public IP addressing
- Instance states (running, stopped)

## Steps Performed

1. Created an EC2 instance using Amazon Linux AMI
2. Generated and downloaded a key pair for secure access
3. Configured Security Group with basic inbound rules
4. Observed instance status and public IP allocation
5. Explored AWS Management Console for instance monitoring

## Outcome

Successfully launched and managed an Amazon EC2 instance using AWS Console. Gained foundational understanding of cloud infrastructure provisioning, instance configuration, and basic networking concepts.

## Skills Gained

- AWS EC2 basics
- Cloud infrastructure fundamentals
- Key pair management
- Security group configuration
- Public IP understanding
