# AWS EC2 Web Server Project

This project demonstrates launching an EC2 instance in AWS and hosting a simple web page using Apache web server.

## Project Objective

To understand the basics of:
- Amazon EC2
- Security Groups
- Key Pairs
- Connecting via SSH
- Installing and configuring a web server

## Steps Performed

1. Created an EC2 instance (Amazon Linux)
2. Configured Security Group to allow HTTP (Port 80) and SSH (Port 22)
3. Connected to the instance using SSH
4. Installed Apache web server
5. Created a sample HTML web page
6. Started and enabled Apache service
